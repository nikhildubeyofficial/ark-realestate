# Plan: Fix Awards Images and Add Testimonial Videos

## Goal
1. Fix the awards section images that are not loading

## Issues Identified

### 1. Awards Section - Images Not Loading
**Root Cause**: The award image file names contain spaces and special characters (parentheses) that are not being properly URL-encoded when used in CSS `background-image` URLs.

**Current implementation**:
```tsx
// Lines 20-50 in page.tsx - file names with spaces/parentheses
"/awards/WhatsApp Image 2026-04-09 at 9.35.45 PM (1).jpeg"
```

**Files in public/awards/**: 28 WhatsApp image files with spaces and parentheses in names

**Available videos**:
